SimpleProxyASB Portable Edition

Описание:
Это portable-версия приложения SimpleProxyASB, которая не требует установки.
Приложение полностью функционально и включает все возможности оригинальной версии:

- Интегрированные поля для ввода логина и пароля
- Локальный прокси-туннель с аутентификацией
- Сохранение и восстановление настроек
- Корректное закрытие с разрывом соединения
- Защита от проблем с подключением после закрытия

Как использовать:
1. Запустите SimpleProxyASB.exe
2. Введите данные подключения (IP, порт, логин, пароль)
3. Нажмите Connect для подключения
4. При закрытии приложения через крестик в правом верхнем углу приложение автоматически разорвет соединение

Важно:
- Приложение сохраняет настройки в файл settings.json в той же папке
- Для корректного закрытия всегда используйте крестик в окне приложения
- При аварийном закрытии приложения настройки прокси могут остаться в системе
