# ASB SimpleProxy

Простое и мощное приложение для управления прокси-серверами в Windows с поддержкой HTTP/HTTPS прокси, обработкой аутентификации и мультиязычным интерфейсом (английский/русский).

## Функции

- Подключение к HTTP/HTTPS прокси с аутентификацией
- Автоматическая конфигурация прокси в реестре Windows
- Локальный прокси-туннель для обработки аутентификации
- Мультиязычный интерфейс (английский/русский)
- Доступна компактная portable версия
- Мониторинг статуса
- Поддержка автоматического переподключения

## Требования

- Windows 10/11
- Python 3.8+
- Права администратора (для изменения реестра)

## Установка

### Из исходного кода

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ASB-MSK/SimpleProxyASB.git
cd SimpleProxyASB
```

2. Установите зависимости:
```bash
pip install customtkinter pyinstaller
```

3. Запустите приложение:
```bash
python proxy_app.py
```

### Portable версия

Скачайте portable версию со страницы [релизов](https://github.com/ASB-MSK/SimpleProxyASB/releases):
- Распакуйте ZIP архив
- Запустите `ASB SimpleProxy.exe`

## Сборка из исходного кода

Для создания portable версии:

```bash
pyinstaller --onefile --windowed --name "ASB SimpleProxy" proxy_app.py
```

Исполняемый файл будет в папке `dist/`.

## Использование

1. Введите IP адрес и порт прокси
2. (Необязательно) Введите логин и пароль для аутентификации
3. Выберите тип прокси (HTTP или HTTPS)
4. Нажмите "Подключить" для установления соединения
5. Нажмите "Отключить" для прекращения использования прокси

## Язык

Нажмите кнопку "RU/EN" в правом верхнем углу для переключения между английским и русским языками.
